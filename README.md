Turf Management System

🚀 Overview
A full-stack Turf Booking System that allows users to browse, book, and manage turf reservations with real-time updates.

 🛠 Tech Stack
- Backend: Spring Boot, Spring MVC, JPA, Hibernate
- Frontend: React.js
- Database: PostgreSQL
- Architecture: Monolithic (Layered)

✨ Features
- Turf listing and booking system
- Membership management
- Real-time booking status updates
- User and Owner dashboards
- RESTful API integration

🔧 Setup
1. Clone repo
2. Configure database
3. Run backend (Spring Boot)
4. Run frontend (React)
